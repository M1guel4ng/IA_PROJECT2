# Frontend (Canvas)

## Correr
1) Inicia el backend (FastAPI) en `http://localhost:8000`.
2) Abre `index.html` con un servidor estático (recomendado).

### Opción A: VSCode Live Server
- Clic derecho en `index.html` -> "Open with Live Server"

### Opción B: Python simple server
```bash
cd frontend
python -m http.server 5173
```
Luego abre: http://localhost:5173

## Controles
- Reset (vale + algoritmo)
- Step / +10
- Play/Pause con velocidad ajustable
